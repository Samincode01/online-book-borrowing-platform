# Book House – Online Book Borrowing Platform

## Purpose

Book House is an online book borrowing platform where users can browse, search, and borrow books from categories like Story, Tech, and Science. Readers can create an account, log in, view book details, borrow books, and manage their profile online.

## Live URL

https://online-book-borrowing-platform-silk.vercel.app

## Key Features

- Browse books by category
- Search books by title
- Filter books by category
- View book details
- Borrow books with quantity tracking
- User registration and login authentication
- Google Sign-In
- Protected routes for authenticated users
- Update user profile
- Toast notifications for user actions
- Fully responsive design
